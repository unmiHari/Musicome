# Musicome
一简单的音乐播放器。

A simple music player.

## 介绍
Musicome 是由 Jetpack Compose 开发的 Android 仿 Apple Music 的音乐播放器。

Musicome is an Android music player developed by Jetpack Compose that mimics Apple Music.

## 功能
当前目标 v1.0：
 - 完善本地播放所有功能
未来目标 v2.0：
 - 插件式网络播放
